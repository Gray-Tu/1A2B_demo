錯誤: 非數字

吃瓜式解法一：自定函式、Try and Except、unicodedata package




















def is_number(string_A):
    """
        檢查 string_A 是否由數字組成
        ref: http://www.runoob.com/python3/python3-check-is-number.html
    """
    try:
        float(string_A)
        return True
    except ValueError:
        pass
        
    try:
        import unicodedata
        unicodedata.numeric(string_A)
        return True
    except (TypeError, ValueError):
        pass
       
    return False

錯誤: 非數字

簡單解：用 字串 method 來判斷, 注意有個點 .

“字串”.isdigit()
“字串”.isdecimal()
“字串”.isnumeric()








MyInput = "012345"
print(MyInput+" 是不是數字", MyInput.isdigit()) #True

MyInput = "A012345"
print(MyInput+" 是不是數字", MyInput.isdigit()) #False

#這些方法不能判斷負數 如 "-1"

錯誤: 有重複

吃瓜式解法：

  • list() 轉換
  • -> set() 去重複
  • -> len() 比較長短








def is_repeat(string_A):
    """
        檢查數字是否重複
    """
    if len(list(string_A)) != len(set(list(string_A))):
        return True
    else:
        return False

錯誤: 有重複

步驟解：

  1. 取出第一個，跟後面的比較
  2. 取出第二個，跟後面的比較
  3. 取出第三個，跟後面的比較
  4. 完

大概像這樣

  1. 取出第一個，跟後面的比較
  1. 取出第二個，跟後面的比較
  1. 取出第三個，跟後面的比較

若都沒有出現一樣的元素，表示沒有重複

讓 迴圈 幫你完成重複的工作

for loop








MyInput = input("請輸入 4位 不重複 的 數字")

for ind in range(len(MyInput)):
    print("第 "+str(ind)+" 個位置是: "+MyInput[ind])
    
# 此處 range() 可以控制執行 for loop 的次數
# 本例次數為 len(MyInput)

步驟比較




























MyInput = "1234"

Flag = 0  #準備一個旗子
#第一次比較
x = 0
for y in range(1, len(MyInput)): #由第二個位開始比較
    if MyInput[x] == MyInput[y]:
        print("有重複")
        Flag = 1  #插個旗子，表示有遇到重複

x = 1
for y in range(2, len(MyInput)): #由第三個位開始比較
    if MyInput[x] == MyInput[y]:
        print("有重複")
        Flag = 1  #插個旗子，表示有遇到重複

x = 2
for y in range(3, len(MyInput)): #由第四個位開始比較
    if MyInput[x] == MyInput[y]:
        print("有重複")
        Flag = 1  #插個旗子，表示有遇到重複

if Flag == 0:
    print("沒重複")
else:
    print("回去重寫")

雙迴圈整合版














MyInput = "1234"

Flag = 0  #準備一個旗子

for x in range(len(MyInput)-1): #只做 0, 1, 2
    for y in range(x+1, len(MyInput)):
        if MyInput[x] == MyInput[y]:
            Flag = 1 #插個旗子，表示有遇到重複

if Flag == 0:
    print("沒重複")
else:
    print("有重複，回去重寫")
